**Layers (Option A shape):** A codify catalogue in SemanticStore schema · B catalogue (operator artifact) · C wire remaining agents' `kg_writer` (per-agent ownership; seal #19 stays clear) · D `kg_query` + findings-as-decorations · E migration + verification. _kg_query/shared-interface PRs + any ADR = per-PR review; per-agent kg_writer wiring = self-merge._
